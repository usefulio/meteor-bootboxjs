###Bootboxjs - Meteor Smart Package w/o Bootstrap 3 dependency

This is a the [bootboxjs bootstrap plugin](http://bootboxjs.com) as a meteor smart package.

###How to use?

In your smart.json:

```
"bootboxjs": {
  "git": "https://github.com/usefulio/meteor-bootboxjs.git",
  "branch": "master"
}
```
